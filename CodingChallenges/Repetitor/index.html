Repetitor

See code on GitHub

With Repetitor, you can repeat a whole theater scene with nobody but a computer (and a good old browser like Chrome or Firefox!). Go practice your memory!

Instructions : choose a character to repeat. Clic Run. Press right arrow to skip the current sentence. Press space to pause

Add another sheet

You can use any Google sheet with the following format :

  • first colon named character and referencing the character who speaks
  • second colon named sentence, which carries the sentence to be spoken
  • third colon named duration is optionnal. The value in second represents the time given to this sentence when it is hidden

The Google sheet must be shared and published on the web. Then, get the URL to that file. Run the program with this URL and use the urlSeet parameter (index.html?urlSheet=URL-to-sheet)
